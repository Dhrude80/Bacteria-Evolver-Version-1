# Bacteria-Evolver-Version-1
Initial version 1 of a Bacteria generation evolver. Mutates and reproduces bacteria indicating survival and reproducing ability based on GC count.

Bacterias are randomly generated and later mutated and undergo reproduction, while having to constantly survive. This is just a simple version 1 with no environmental factors, diseases, immunity consideration, etc.

---

## Features

- Random DNA string generation
- Survival Probability
- Data tracking and copying
- Matplot Lib plotting

---

## Technologies used

- Python
- Maptlot Lib
- Numpy
- Dictionaries

Install required libraries before running.
